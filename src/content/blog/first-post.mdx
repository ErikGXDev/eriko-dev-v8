## What's Astro?

Astro is a Javascript framework like Next, Nuxt or SvelteKit.
It's focused on performance by encouraging the use of static pages and is useful for content-heavy websites like blogs, documentation or portfolios.
Astro also allows the use of multiple UI frameworks.
This means you can integrate React, Svelte and Vue components into one page!
